## 🎮 概述

這是 Journey of Kindness 的第七關，結合經典的**電車難題**（Trolley Problem）與 **Alpha-Beta Pruning** 演算法概念。

玩家面對一個道德困境：一輛失控的電車正衝向五個家庭，你可以拉動開關讓電車轉向，但另一條軌道上站著一位長者...

**但這不只是二選一！** 如果你發現「第三條路」，就能解鎖隱藏的 C 結局 — 慈悲的奇點。

---

## 📚 AI 演算法：Alpha-Beta Pruning

**Russell & Norvig Chapter 5.3**

Alpha-Beta Pruning 是一種對抗搜尋優化技術，用於減少 Minimax 演算法需要評估的節點數。

在本遊戲中，我們用它來比喻：
- **剪枝** = 排除不可能的選項
- **評估函數** = 道德權重的計算
- **博弈** = 不同 AI 哲學之間的對抗
